# 帕斯卡画廊内容分类

本仓库曾用于归档材料分类。当前公开网站采用无版权风险策略，只保留可公开的概括性服务说明。

## 分类

| 分类编号 | 中文名称 | 用途 |
| --- | --- | --- |
| `works` | 作品 | 仅在获得授权后公开作品记录 |
| `exhibitions` | 展览 | 仅在获得授权后公开展览信息 |
| `artists` | 艺术家 | 仅在获得授权后公开艺术家简介、肖像和履历 |
| `texts` | 文献 | 仅在获得授权后公开策展文章、访谈或研究文字 |
| `media` | 媒体资料 | 仅在获得授权后公开海报、活动照片、视频和下载资料 |

## 数据规则

- 当前公开页面读取 `data/xu-services.json`，不读取图片素材。
- 未确认授权的客户名称、项目来源文件名、图片、截图、PPT 原文和长摘录不得写入公开数据。
- 如需新增图片，必须先确认授权，再放入 `assets/` 并记录来源。
- 真实归档材料准备好之前，只使用概括性服务描述。
